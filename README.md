# なにわモーター 統合ツール 完全版

## できること
- 素材買取アプリ
- ギャングカラー検索
- 修理/色替え 作業カウンター
- ギャングデータ登録、編集、削除
- ホーム画面から切り替え
- ローカル保存で、ホームに戻っても内容保持

## GitHub Pages 反映方法
1. ZIPを解凍
2. GitHub の今のリポジトリを開く
3. `index.html` を上書きアップロード
4. 必要なら `README.md` も上書き
5. Commit changes
